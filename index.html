Intro:

This year I have been a part of the Community Software Group at Otago Polytechnic. This group was set up to allow community groups access to software who could otherwise not afford it. The three main tasks that we have done this project are:

  1. Pathmapper - An android mobile application for the Dunedin Botanical Gardens
  2. Visual Scan Suite - An android tablet application designed to aid in the diagnosis of visual impairment following a brain injury. This was done in collaboration with Dr Mary Butler of the Occupational Therapy School
  3. ASP.net Frontend / SQL Backend database - We designed an ASP.net API front end interface for the Botanical Gardens which talks to the database and allows the user to upload images/create QR codes and add to the database.
During the semester I was mainly involved in the on going development of Visual Scan Suite and also the ASP.net frontend and SQL database backend portion of the project.
